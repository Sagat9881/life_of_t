## Задача

Спроектировать **техническую спецификацию HTML Overview-страницы** для раздела метрик на Project Status Dashboard.

## Контекст

- Страница — полностью статичный HTML (без фреймворков, только vanilla JS + CSS)
- Данные загружаются из `metrics.json` (генерируется пайплайном, лежит рядом с HTML)
- Страница открывается из GitHub Pages
- Дизайн: минималистичный pixel-art стиль, соответствующий визуальному стилю игры Life of T

## Цель

Создать спецификацию `docs/specs/STATUS_PAGE_OVERVIEW_SPEC.md` описывающую:
1. Структуру HTML-документа (layout, навигация, секции)
2. Компоненты для верхнеуровневых метрик (карточки)
3. Компоненты детализации (drilldown-панели)
4. Механизм навигации Overview → Детализация
5. Формат отображения каждого типа метрик

## Требования к Overview-странице

### Структура документа

```
index.html
├── <header>  — название проекта, дата генерации, ссылка на Assets Gallery
├── <nav>     — табы: Overview | Assets Gallery
├── <main id="overview">
│   ├── .metrics-grid  — 5 карточек верхнеуровневых метрик
│   └── .drilldown-panel  — панель детализации (скрытая по умолчанию)
└── <footer>  — версия, SHA последнего коммита
```

### Карточки метрик (`.metric-card`)

Каждая карточка содержит:
- Иконка категории (emoji или SVG-иконка в pixel-art стиле)
- Название метрики
- Главное значение (большой шрифт)
- Мини-прогресс-бар или статус-бейдж
- При клике: открывается `.drilldown-panel` с детализацией

### Детализация (`.drilldown-panel`)

- Панель разворачивается под карточкой (accordion) или сайдбар
- Содержит список из 3–5 sub-метрик с:
  - Названием
  - Значением
  - Визуализацией (прогресс-бар / бейдж / число)
- Кнопка закрытия / клик на карточку повторно закрывает

### Визуализация типов метрик

| Тип | Компонент |
|-----|-----------|
| Процент (0–100%) | горизонтальный прогресс-бар с цветовой индикацией (red/yellow/green) |
| Счётчик (число) | просто число, крупный шрифт |
| Статус (success/failure/pending) | цветной бейдж |
| Распределение по категориям | мини-столбчатая диаграмма (CSS-only) |
| Дата/время | relative time ("2 hours ago") |

### Цветовая индикация прогресса

- 0–33%: `#e74c3c` (красный)
- 34–66%: `#f39c12` (жёлтый)
- 67–100%: `#27ae60` (зелёный)
