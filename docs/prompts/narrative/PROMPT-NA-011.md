## Что создать

Файл `narrative/plans/arc-01-baseline.md` со следующими разделами:

### 1. Цель арки
- Центральная нарративная эмоция
- Ключевой вопрос арки («Что в итоге понимает Татьяна?»)
- Жанровые рамки (светлый ромком, без тёмных финалов)

### 2. Структура актов
- Акт I: Завязка — ключевые события, квесты, первые конфликты
- Акт II: Развитие — основные конфликты, NPC-взаимодействия, повороты
- Акт III: Развязка — кульминация, концовки, эмоциональный финал

### 3. Квесты по актам
- Таблица: Квест → Акт → Цель в арке → Эмоция
- Все 15 квестов должны быть распределены

### 4. Конфликты
- Таблица: Конфликт → Акт → Стороны → Путь к разрешению
- Все 10 конфликтов

### 5. Концовки
- Таблица: Концовка → Условие достижения → Эмоциональный тон
- Все 10 концовок

### 6. NPC-роли
- Краткое описание вклада каждого ключевого NPC в арку

### 7. Ограничения жанра
- Список «красных линий» из `narrantor-skill.md` §3, применённых к данной арке
