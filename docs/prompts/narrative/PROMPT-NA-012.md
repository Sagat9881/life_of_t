---

## Задача

Создай шаблон ревью квеста: файл `narrative/reviews/quest-review-template.md`.

---

## Контекст

- Каждый квест в `game-content/.../narrative/quests/` должен проходить стандартное ревью перед финализацией.
- Без шаблона ревью проводятся неодинаково → риск пропуска нарративных ошибок.
- Шаблон используется Narrator при ревью любого квеста.

---

## Что создать

Файл `narrative/reviews/quest-review-template.md` со следующей структурой:

### Заголовок
```
Quest ID: ___
Quest Name: ___
Date: ___
Reviewer: Narrator
```

### 1. Жанровый чеклист (по narrantor-skill.md §3)
- [ ] Финал квеста — позитивный или нейтральный (не депрессивный)
- [ ] Нет жестокости / насилия / безнадёжности как нормы
- [ ] Конфликт внутри квеста имеет путь к разрешению
- [ ] Тон соответствует «светлому ромкому»

### 2. Чеклист фактов (по tatyana-facts.md)
- [ ] Все факты о Татьяне согласованы с `narrative/world/tatyana-facts.md`
- [ ] Не нарушены «каноны и запреты» из tatyana-facts.md
- [ ] Реальные персонажи (муж, родители, животные) изображены корректно

### 3. Чеклист структуры спецификации
- [ ] Присутствует `event_triggers/start.md`
- [ ] Присутствует `event_triggers/success.md`
- [ ] Присутствует `event_triggers/fail.md`
- [ ] Ветвления (`branch-*.md`) описаны, если применимо

### 4. Чеклист NPC
- [ ] Все упомянутые NPC имеют спецификации в `narrative/specs/npc/`
- [ ] Образы NPC согласованы с их спецификациями
- [ ] Нет токсичных NPC без разворота к теплу

### 5. Итог
```
Статус: APPROVED / NEEDS_REVISION
Комментарий: ___
```
