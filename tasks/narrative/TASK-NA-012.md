## Описание

Отсутствует стандартный шаблон ревью для квестов (`narrative/reviews/`). Без шаблона каждый квест проверяется по-разному, что приводит к пропускам нарративных противоречий и нарушений жанровых ограничений.

Требуется создать шаблон-чеклист, по которому Narrator проверяет каждый квест перед финализацией спецификации.

**Промт для выполнения:** `docs/prompts/narrative/PROMPT-NA-012.md`

---

## Входные артефакты

- `narrantor-skill.md` — раздел 3 «Жанровые и смысловые ограничения», раздел 6 «Методология SDD»
- `narrative/world/tatyana-facts.md` — **требуется** (блокирующая зависимость от TASK-NA-010)
- `game-content/life-of-t/src/main/resources/narrative/quests/` — примеры существующих квестов

---

## Выходные артефакты

- `narrative/reviews/quest-review-template.md` — шаблон чеклиста ревью квеста

### Обязательные разделы шаблона:

1. **Заголовок** — ID квеста, имя квеста, дата ревью, ревьюер
2. **Жанровый чеклист** — проверка соответствия «светлому ромкому» (по `narrantor-skill.md` §3)
3. **Чеклист фактов** — соответствие `narrative/world/tatyana-facts.md`
4. **Чеклист структуры** — наличие start / success / fail триггеров
5. **Чеклист NPC** — согласованность NPC-образов с их спецификациями
6. **Итог** — APPROVED / NEEDS_REVISION с комментарием
