## Описание

Отсутствует `narrativeplans/content-plan.md` — основной документ, определяющий карту нарративного контента: какие квесты, NPC и события входят в каждую игровую фазу, как они связаны между собой и в какой последовательности должны быть реализованы.

Без content-plan невозможно приоритизировать оставшийся нарративный объём (15 квестов, 10 конфликтов, 10 концовок) и поставить задачи на Java Developer по порядку загрузки данных.

## Выходные артефакты

- `narrativeplans/content-plan.md` — карта нарративного контента

### Обязательные разделы файла:
1. **Фазы игры** — список фаз (например: Начало, Развитие, Кульминация, Финал)
2. **Контент по фазам** — таблица: квест/NPC/конфликт/концовка → фаза
3. **Зависимости** — цепочки unlock-условий между квестами
4. **Приоритет реализации** — очерёдность задач для Narrator и Java Developer
5. **Покрытие концовок** — какие концовки закрывают какие нарративные дуги
