## Описание

Реализовать компонент `.drilldown-panel` и логику навигации между карточками метрик. Дополняет TASK-FE-050 — реализует раскрытие/закрытие, анимацию, sub-метрики и bar-chart.

## Требования

1. **HTML-структура** по Разделу 3.1 спеки — `.drilldown-panel__header`, `.drilldown-panel__list`, `.drilldown-panel__chart`
2. **Анимация** по Разделу 3.2:
   - Открытие: `max-height 200ms ease-out`, `opacity 200ms`
   - Закрытие: `150ms ease-in`
   - Смена категории: fade-out 100ms → смена содержимого → fade-in 100ms
3. **Позиция** в grid: `grid-column: 1 / -1` (Раздел 3.3)
4. **Sub-метрики** для всех 5 категорий по таблицам Раздела 3.4
5. **Bar-chart** CSS-only для `byRole` (taskBoard) — столбики через `height` в %
6. **Клавиатурная навигация**: `Escape` закрывает drilldown, фокус возвращается на карточку
7. Кнопка `[✕]` в заголовке — закрывает панель

## Критерии приёмки

- [ ] Drilldown открывается под активной карточкой
- [ ] Анимация плавная (no jank) при открытии/закрытии
- [ ] Смена карточки без мигания: fade-out → смена → fade-in
- [ ] `Escape` закрывает drilldown
- [ ] Sub-метрики соответствуют таблицам §3.4 для каждой категории
- [ ] Bar-chart отображает % по ролям для taskBoard
- [ ] `grid-column: 1 / -1` работает на всех breakpoints
- [ ] Кнопка ✕ закрывает панель

## Метрики, затрагиваемые задачей

- Все 5 групп (TBH, BC, AC, NC, CCH) — визуализация sub-метрик в drilldown
