## Описание

Спроектировать полную спецификацию метрик проекта Life of T для автоматически генерируемой HTML-страницы статуса. Определить 5 верхнеуровневых групп метрик, детализацию каждой, источники данных, формулы расчёта и JSON-схему выходного файла `metrics.json`.

## Критерии приёмки

- [ ] Файл `docs/specs/PROJECT_METRICS_SPEC.md` создан и содержит все 5 групп метрик
- [ ] Каждая метрика имеет: ID, источник данных, формулу, формат отображения
- [ ] JSON-схема `metrics.json` полностью задокументирована в Разделе 4
- [ ] Глоссарий содержит все ключевые термины (Раздел 1)
- [ ] Схема сбора данных описана (Раздел 3)
- [ ] Нет метрик без явного источника данных
- [ ] Созданы задачи TASK-BE-018, TASK-BE-019, TASK-BE-020

## Метрики, затрагиваемые задачей

- TBH (Task Board Health)
- BC (Backend Completeness)
- AC (Assets Coverage)
- NC (Narrative Coverage)
- CCH (CI/CD Health)
