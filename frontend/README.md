# Life of T - Frontend

Уютный life-sim симулятор жизни Татьяны в Telegram Mini App.

## Технологии

- **React 18** + **TypeScript 5**
- **Vite 5** - быстрая сборка
- **Zustand** - state management
- **Framer Motion** - анимации
- **Telegram WebApp SDK** - интеграция
- **CSS Modules** - стилизация

## Запуск

```bash
# Установка зависимостей
npm install

# Разработка
npm run dev

# Сборка
npm run build

# Превью
npm run preview
```

## Структура

```
src/
├── api/            # API клиент
├── components/     # Компоненты
├── hooks/          # Custom hooks
├── pages/          # Экраны
├── store/          # Zustand store
├── types/          # TypeScript types
├── utils/          # Утилиты
└── styles/         # Стили
```
